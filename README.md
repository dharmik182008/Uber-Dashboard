# 🚖 Uber Ride Analytics Dashboard | Power BI

An interactive **Power BI dashboard** built to analyze Uber ride booking data, providing insights into bookings, revenue, customer behavior, vehicle performance, payment methods, and operational metrics.

---

## 📊 Dashboard Overview

This dashboard helps visualize Uber ride data through interactive reports and KPIs, enabling better business decisions and performance monitoring.

### Key Metrics

- ✅ Total Completed Bookings: **93K**
- ❌ Lost Bookings: **57K**
- 💰 Total Revenue: **51.85M**
- 📍 Total Distance Travelled: **2.51M**
- 🚗 Average Ride Distance: **24.64 km**

## 🚀 How to Use

1. Clone this repository.

```bash

```

2. Open the `.pbix` file using **Power BI Desktop**.

3. Explore the dashboard using filters and slicers.

## 📌 Insights

- Auto generated the highest revenue among all vehicle types.
- UPI is the most preferred payment method.
- Revenue remains relatively consistent across months.
- Peak ride demand occurs during evening hours.
- Regular riders contribute significantly to total revenue.
